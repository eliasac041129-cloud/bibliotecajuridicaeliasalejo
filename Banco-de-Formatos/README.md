# 📁 Banco de Formatos

Plantillas **reales y listas para practicar** de los documentos que un abogado corporativo redacta a
diario. No son teoría: son **modelos operativos** que aprenderás a llenar, negociar y defender.

> **⚠️ Aviso.** Son **modelos educativos** para adaptar a cada caso concreto; **no** sustituyen el
> criterio profesional. Los campos entre `[CORCHETES]` se completan; las notas *en cursiva* explican el
> porqué de cada cláusula. **⟳** Todo fundamento legal citado debe **reverificarse vigente** en su código
> ([`../fuentes-legales/`](../fuentes-legales/)) antes de usarse en un asunto real.

## Cómo usar este banco (método)

1. **Lee el formato completo** una vez, con el tratado de la materia al lado.
2. **Llénalo** con un caso hipotético (o real anonimizado).
3. **Justifica cada cláusula:** ¿qué riesgo cubre?, ¿qué pasa si se omite?
4. **Verifica** cada artículo citado en [`../fuentes-legales/`](../fuentes-legales/) (disciplina ⟳).
5. **Negócialo:** identifica qué cláusulas defiende el comprador y cuáles el vendedor.
