# Nivel I · Fundamentos

> ⟳ **Apóstrofe de vigencia — léela cada vez.** El Derecho cambia sin avisar: un artículo puede mudar de número, de redacción o quedar **derogado** de un día para otro. El símbolo **⟳** que aparece tras cada artículo citado en este libro significa una sola cosa: *«¿sigue vigente —y con este mismo número— hoy? No lo cites de memoria ni desde este libro: **reitéralo en su código vigente**»* (textos oficiales en [`../../fuentes-legales/`](../../fuentes-legales/)). Caso real: el **art. 390 del CPF** que este proyecto cotejó aparece hoy como **«Derogado»**. Recuerda: un **✅** dice que el dato fue verificado *palabra por palabra a la fecha de su fuente*; la **⟳** te avisa que esa fecha ya quedó atrás y que la última palabra la tiene el código, no el libro.


> **Objetivo:** construir la *arquitectura mental* del Derecho. Al terminar este nivel,
> el lector comprenderá **por qué** existen las instituciones jurídicas, no solo qué
> dicen las leyes.

Este es el nivel más importante a largo plazo: sin cimientos conceptuales sólidos, todo
lo demás se memoriza y se olvida. Aquí se forja el **criterio**.

## Tratados del nivel

1. [Derecho Romano](./01-Derecho-Romano.md) — la gramática del derecho privado.

9. Historia del Pensamiento Jurídico y Económico.
10. Economía para juristas.
11. Argumentación Jurídica.
12. Lógica Jurídica.
13. Metodología Jurídica.

## Por qué este orden

Empezamos en **Roma** porque el derecho privado mexicano (y la mitad del mundo) habla
latín conceptual: *obligatio*, *dominium*, *persona*, *res*, *contractus*. Quien domina
las categorías romanas lee el Código Civil como quien conoce el alfabeto. Sobre esa base
se montan la teoría del acto jurídico, las obligaciones y los contratos, que son la
materia prima de toda operación corporativa.
